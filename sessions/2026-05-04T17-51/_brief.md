# 📋 작업 브리프

**원 명령:** 깃허브 동기화

## 요약
회사 내부의 모든 핵심 지식(회사 정체성, 목표, 의사결정 로그, 개인 메모리)을 최신 상태로 GitHub에 동기화합니다. 이는 모든 팀원이 동일한 기준을 갖도록 보장하는 필수 과정입니다.

## 분배
- **💻 Developer**: 회사 핵심 저장소(goals.md, identity.md, decisions.md, ceo_memory.md)의 모든 최신 변경 사항을 GitHub의 메인 브랜치에 푸시(Push)하고, 모든 API 연동 상태를 점검하여 데이터 파이프라인의 무결성을 확인하시오.
- **📱 Secretary**: GitHub 동기화 작업이 성공적으로 완료되었음을 확인하고, '지난 의사결정 로그' 및 'CEO 개인 메모리'에 동기화 완료 기록을 명시하며, 관련 내용을 모든 팀원에게 텔레그램 보고서 형식으로 요약 보고하시오.
