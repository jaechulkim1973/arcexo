# 📋 작업 브리프

**원 명령:** 깃허브 동기화

## 요약
회사의 핵심 지식과 데이터를 Git 저장소에 동기화하고 무결성을 최종 검증하는 작업입니다. 모든 팀이 최신, 신뢰할 수 있는 정보를 바탕으로 다음 운영 단계로 넘어갈 준비를 합니다.

## 분배
- **💻 Developer**: 모든 코어 문서(Goals, Identity, Decisions 등)와 데이터 파이프라인의 Git 동기화가 완료되었는지 최종적으로 재검증하고, 모든 항목의 무결성(Integrity) 보고서를 작성하여 공유하라. 특히 데이터 스키마와 API 연결 상태에 문제가 없는지 꼼꼼히 체크하라.
- **📱 Secretary**: 개발자로부터 받은 동기화 무결성 보고서를 바탕으로, 현재까지의 모든 최신 데이터를 취합하고, 다음 운영 보고서 작성을 위한 운영 준비 상태를 공식적으로 팀에 브리핑하라. 그리고 보고서 작성에 필요한 다음 액션 아이템을 목록화하여 CEO에게 보고할 준비를 하라.
