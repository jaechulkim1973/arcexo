## 💎 1. 디자인 시스템 가이드라인 (Brand Guideline)

'금기반'의 신뢰도를 구축하기 위해, 가볍고 트렌디한 느낌보다는 **역사적 깊이, 안정감, 그리고 전문성**을 강조하는 비주얼 언어를 채택합니다.

### 1.1. 컬러 팔레트 (Color Palette)

| 역할 | 색상명 | Hex Code | 사용 목적 | 비고 |
| :--- | :--- | :--- | :--- | :--- |
| **Primary (신뢰)** | 딥 네이비 (Deep Navy) | `#0A1931` | 헤더, 핵심 버튼, 로고, 배경의 무게감을 줄 때. (금융권의 신뢰감 상징) | 주 색상 (Main Color) |
| **Secondary (골드)** | 앤티크 골드 (Antique Gold) | `#B8860B` | 강조, 성공/검증 완료 표시, 핵심 CTA 버튼. (금(Gold)의 가치 상징) | 액센트 색상 (Accent Color) |
| **Tertiary (안정)** | 라이트 그레이 (Light Gray) | `#F4F6F8` | 배경, 카드 배경, 구분선. (가독성 확보 및 세련된 여백) | 배경 색상 |
| **Neutral (텍스트)** | 차콜 블랙 (Charcoal Black) | `#333333` | 본문 텍스트, 레이블. (어두운 네이비보다 부드러우면서도 가독성이 높음) | 텍스트 색상 |

### 1.2. 타이포그래피 (Typography)

*   **Headline (헤드라인):** Pretendard Bold 또는 Noto Sans KR Bold (가독성이 높으면서도 무게감 있는 산세리프 계열).
    *   **활용:** 페이지 제목, 섹션 구분.
*   **Body (본문):** Pretendard Regular 또는 Noto Sans KR Regular.
    *   **활용:** 설명 텍스트, 입력 필드, 데이터 테이블.
    *   **규칙:** 본문 텍스트는 16px 이상, 줄 간격(Line-height)은 1.6을 유지하여 장문 가독성을 극대화합니다.

### 1.3. 비주얼 시스템 및 컴포넌트

1.  **아이콘:** 단색(Monochrome)의 **라인 아이콘(Line Icon)** 사용. (예: 자물쇠, 체크마크, 화살표). 아이콘을 통해 '보안'과 '완료' 상태를 명확히 전달합니다.
2.  **카드 디자인:** 둥근 모서리(Border-radius: 8px)를 가진 '라이트 그레이' 배경의 카드 컴포넌트를 사용하여, 각 기능(검증, 신청, 결과)을 명확히 분리합니다.
3.  **성공/경고 피드백:**
    *   **성공:** 배경에 옅은 녹색을 사용하고, 앤티크 골드 색상의 체크마크를 중앙에 배치합니다.
    *   **오류:** 배경에 옅은 빨간색을 사용하고, 딥 네이비 색상의 경고 아이콘을 배치합니다.

## 🖥️ 2. 핵심 사용자 플로우 와이어프레임 (Wireframe)

핵심 플로우를 3단계로 나누어 구조화하고, 각 단계별 주요 요소와 레이아웃을 정의합니다.

### 📍 Flow 1: 랜딩 페이지 및 회원가입 (Trust Establishment)

**목표:** 사용자가 즉각적으로 '신뢰'와 '가치'를 인지하게 한다.
**레이아웃:** 2단 구조 (좌측: 가치 설명, 우측: CTA/가입 폼)

| 영역 | 구성 요소 | 디자인 규칙 | 내용 강조 (금기반) |
| :--- | :--- | :--- | :--- |
| **Hero Section** | **[H1]** 금기반 자산 검증 및 자금 조달 플랫폼 <br> **[Body]** 공신력 있는 검증 시스템으로 자산의 투명성을 확보하세요. <br> **[CTA]** 무료 검증 시작 (앤티크 골드 버튼) | 딥 네이비 배경에 앤티크 골드 텍스트를 사용하여 고급스러움을 극대화. | '공신력', '투명성' 등의 단어에 골드 색상 적용. |
| **Feature Card** | 3~4개의 핵심 기능 카드 (검증, 자금 조달, 리스크 분석) | 라이트 그레이 배경의 카드 3열
